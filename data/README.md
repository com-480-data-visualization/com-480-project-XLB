# Data

Raw CSV files are NOT committed to this repository (too large).
See the main README for full download instructions.

## Files

| File | Size | Purpose |
|------|------|---------|
| movies_metadata.csv | ~34MB | Main file: budget, revenue, genres, release dates |
| credits.csv | ~40MB | Cast and crew JSON for director/actor analysis |
| ratings.csv | ~710MB | Full 26M ratings — use on cluster/powerful machine |
| ratings_small.csv | ~2MB | 100k ratings — use for local EDA |
| keywords.csv | ~5MB | Plot keywords (optional enrichment) |
| links.csv | ~5MB | Bridge: MovieLens ID ↔ TMDB ID (full) |
| links_small.csv | <1MB | Bridge: MovieLens ID ↔ TMDB ID (subset) |
