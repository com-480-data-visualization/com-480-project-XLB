# Data

This directory contains all raw and processed data files for the project.

---

## Structure

```
data/
├── raw/                        # Original files from Kaggle — do not edit
│   ├── movies_metadata.csv
│   ├── credits.csv
│   ├── keywords.csv
│   ├── ratings.csv
│   ├── ratings_small.csv
│   ├── links.csv
│   └── links_small.csv
└── processed/                  # Generated by eda/preprocessing.ipynb
    └── movies.csv
```

---

## Raw files

### `movies_metadata.csv`
One row per movie. The main metadata file.

| Column | Type | Description |
| --- | --- | --- |
| `id` | int | TMDb movie ID (primary key) |
| `title` | str | Movie title |
| `original_title` | str | Title in original language |
| `original_language` | str | ISO 639-1 language code (e.g. `en`, `fr`) |
| `release_date` | str | Release date (YYYY-MM-DD, some malformed) |
| `budget` | float | Production budget in USD — zeros represent missing data |
| `revenue` | float | Box office revenue in USD — zeros represent missing data |
| `runtime` | float | Runtime in minutes |
| `popularity` | float | TMDb popularity score |
| `vote_average` | float | TMDb average rating (0–10) |
| `vote_count` | float | Number of TMDb votes |
| `genres` | str | Nested JSON list of genre objects |
| `production_countries` | str | Nested JSON list of country objects |
| `production_companies` | str | Nested JSON list of company objects |
| `spoken_languages` | str | Nested JSON list of language objects |
| `belongs_to_collection` | str | Collection metadata (if part of a franchise) |
| `overview` | str | Plot summary |
| `tagline` | str | Promotional tagline |
| `status` | str | Release status (e.g. Released, Post Production) |
| `adult` | bool | Adult content flag |
| `video` | bool | Video release flag |
| `imdb_id` | str | IMDb identifier |
| `homepage` | str | Official website URL |
| `poster_path` | str | TMDb poster image path |

**Shape:** 45,466 rows × 24 columns

---

### `credits.csv`
One row per movie. Cast and crew stored as nested JSON strings.

| Column | Type | Description |
| --- | --- | --- |
| `id` | int | TMDb movie ID (foreign key) |
| `cast` | str | Nested JSON list of cast members with name, character, and billing order |
| `crew` | str | Nested JSON list of crew members with name, job, and department |

**Shape:** 45,476 rows × 3 columns

---

### `keywords.csv`
One row per movie. Thematic tags from TMDb.

| Column | Type | Description |
| --- | --- | --- |
| `id` | int | TMDb movie ID (foreign key) |
| `keywords` | str | Nested JSON list of keyword objects |

**Shape:** 46,419 rows × 2 columns

---

### `ratings.csv`
One row per user rating. The full MovieLens ratings table.

| Column | Type | Description |
| --- | --- | --- |
| `userId` | int | Anonymous user identifier |
| `movieId` | int | MovieLens movie ID (join via `links.csv`) |
| `rating` | float | Rating on a 0.5–5 scale (half-star increments) |
| `timestamp` | int | Unix timestamp of the rating |

**Shape:** 26,024,289 rows × 4 columns

---

### `ratings_small.csv`
A 100,000-rating subset of `ratings.csv`, useful for quick iteration.

**Shape:** 100,004 rows × 4 columns

---

### `links.csv`
One row per movie. ID mapping between MovieLens, TMDb, and IMDb.

| Column | Type | Description |
| --- | --- | --- |
| `movieId` | int | MovieLens movie ID |
| `imdbId` | int | IMDb movie ID |
| `tmdbId` | float | TMDb movie ID (foreign key to `movies_metadata.csv`) |

**Shape:** 45,843 rows × 3 columns

---

### `links_small.csv`
A subset of `links.csv` corresponding to `ratings_small.csv`.

**Shape:** 9,125 rows × 3 columns

---

## Processed files

Generated by running `eda/preprocessing.ipynb`. Do not edit manually, re-run the notebook to regenerate.

### `movies.csv`
One row per movie. The main clean dataset used for all visualizations.

| Column | Type | Description |
| --- | --- | --- |
| `id` | int | TMDb movie ID |
| `title` | str | Movie title |
| `year` | float | Release year |
| `decade` | float | Release decade (e.g. 1990) |
| `release_date` | datetime | Full release date |
| `budget` | float | Production budget in USD (NaN if unknown) |
| `revenue` | float | Box office revenue in USD (NaN if unknown) |
| `runtime` | float | Runtime in minutes |
| `original_language` | str | ISO 639-1 language code |
| `popularity` | float | TMDb popularity score |
| `vote_average` | float | TMDb average rating (0–10) |
| `vote_count` | float | Number of TMDb votes |
| `genres_list` | list | Parsed list of genre names e.g. `['Action', 'Comedy']` |
| `countries_list` | list | Parsed list of production country names |
| `companies_list` | list | Parsed list of production company names |
| `languages_list` | list | Parsed list of spoken language names |
| `director` | str | Director name (from `credits.csv`) |
| `top_cast` | list | Top 3 billed actors e.g. `['Tom Hanks', 'Meg Ryan', ...]` |
| `movielens_avg_rating` | float | Average MovieLens rating (0.5–5) |
| `movielens_vote_count` | int | Number of MovieLens ratings |
| `rating_min` | float | Minimum MovieLens rating received |
| `rating_median` | float | Median MovieLens rating |
| `rating_max` | float | Maximum MovieLens rating received |
| `rating_std` | float | Standard deviation of MovieLens ratings (polarization proxy) |
| `ratings_list` | list | Full list of individual MovieLens ratings |

**Shape:** 45,697 rows × 25 columns

## Notes

- Raw files are not tracked by git (listed in `.gitignore`) due to their size. Download them using the instructions in the main `README.md`.
- `budget` and `revenue` zeros in the raw file represent missing data, not actual zero values. They are replaced with `NaN` during preprocessing.
- The financial subset (films with both valid `budget` and `revenue`) covers 5,417 films, roughly 12% of the full dataset.
